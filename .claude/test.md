Ты помогаешь проверить Lua скрипт для Roblox executor на ошибки и баги.

# Задача

1. Найди последний .lua файл, с которым работал пользователь (через git status, недавно измененные файлы, или спроси у пользователя)
2. Прочитай содержимое скрипта
3. Проверь на:
   - Синтаксические ошибки Lua
   - Типичные ошибки Roblox API (устаревшие функции, неправильное использование)
   - Логические баги (nil checks, индексация, циклы)
   - Проблемы безопасности для executor окружения
   - Производительность (лишние циклы, утечки памяти)
4. Если нашел ошибки - исправь их автоматически
5. Выведи краткий отчет: что проверено, что исправлено

# Формат ответа

Отвечай на русском языке. Будь кратким и конкретным.
